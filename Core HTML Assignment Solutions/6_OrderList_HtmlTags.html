Order List of Each HTML Tags and its Uses in HTML5

  1. head: It is Something about Meta data (Data about the Data) So this is the Data about the webpage.
  2. body: It is where we used to write our Functionality in HTML Lanaguage which can be Appeared on the Webpage.
  3. h1: It is a High Priority Heading Tag in HTML.
  4. p: It is a paragraph Tag in HTML used to Write paragraph Summary in Webpage.
  5. div: It is used to Grouping the HTML Tags Together like Container Manner.
