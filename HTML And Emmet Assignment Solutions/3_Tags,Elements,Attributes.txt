3. Write a short note on tags, elements, and attributes along with relevant examples ?

In HTML, Tags, elements and Attributes are very Helpfull to design the Structure and Content of the webpage.

Tags:
-----
i)HTML tags are the keywords that are used to produce web pages in various formats.
ii)In HTML, tags are used to mark up and define the content of a web page, They are enclosed in angle brackets and usually come in pairs. 
iii)Opening tags and closing tags are found in the majority of tags, The concluding tags contain a forward slash(/), while the language of the beginning tags is the same. Certain tags don't need to be closed.
iv)There are many different types of HTML tags available, each with its own specific purpose. 

eg: <p>Iam Learning Full Stack Web Development By Physics Wallah.</p>

<p>: Opening Tags
</p>: Closing Tags 

Elements:
---------
An HTML element is defined by a starting tag. If the element contains other content, it ends with a closing tag, where the element name is preceded by a forward slash.

eg: <p>Iam Learning Full Stack Web Development By Physics Wallah.</p>

<p>: Opening Tags
</p>: Closing Tags 
The content between Opening and Closing Tags is called Elements.

Attributes:
-----------
i)All HTML elements can have attributes.
ii)Attributes provide additional information about elements.
iii)Attributes are always specified in the start tag.
iv)Attributes usually come in name/value pairs like: name="value".

eg: <a href="https://www.Google.com">Go To Google</a>
